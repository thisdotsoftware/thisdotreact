# Colours

All of our colours are saved in CSS variables, allowing us to support multiple
brands and special colour schemes such as dark mode and high-contrast.

See the <LinkTo kind="Guides/CSS Variables" story="page">CSS Variables guide</LinkTo>
for instructions on adding the CSS variables to your page.

## Neutral

<ColorBlock colours={["--color-black", "--color-white"]} />

export const ColorBlock = ({ colours }) => (
  <div
    style={{
      display: "flex",
      flexWrap: "wrap",
      gap: "20px",
      marginBottom: "20px",
      marginTop: "15px",
    }}
  >
    {colours.map((colour) => (
      <div key={colour}>
        <div
          style={{
            width: "100%",
            minWidth: "150px",
            height: "45px",
            background: `var(${colour})`,
            marginBottom: "5px",
            border:
              colour === "--color-white"
                ? "1px solid var(--color-grey-300)"
                : "0px",
          }}
        />
        <span>{colour}</span>
      </div>
    ))}
  </div>
);
