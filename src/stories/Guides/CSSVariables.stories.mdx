# CSS Variables

<Description of={ThemeVariables} />

<ArgsTable of={ThemeVariables} />

All of our colours are saved in CSS variables, allowing us to support multiple
brands and special colour schemes such as dark mode and high-contrast.

To add the CSS variables to your page, use the `ThemeVariables` component.

<Source
  language="js"
  code={dedent`
    import { ThemeVariables } from "@thisdotsoftware/react"
    
    <ThemeVariables />
  `}
/>

See the <LinkTo kind="Foundations/Colour" story="page">Colour</LinkTo> and
the <LinkTo kind="Foundations/Typography" story="page">Typography</LinkTo> stories for a
list of CSS variables.
