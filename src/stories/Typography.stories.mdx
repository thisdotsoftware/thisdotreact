# Typography

We have React components for every element of typography - including headings,
paragraphs, links, lists, and more. Each component reads from CSS variables set
on the page, allowing us to theme the components.

- [CSS variables]

- [List] and [ListItem]

## CSS Variables

**Family**

- `--font-family`

**Weights**

- `--font-weight-normal`
- `--font-weight-bold`

**Font sizes**

- `--font-size-heading-1`
- `--font-size-heading-2`
- `--font-size-heading-3`
- `--font-size-heading-4`
- `--font-size-heading-5`
- `--font-size-body-16`
- `--font-size-body-14`
- `--font-size-body-small`

**Line heights**

<Alert>
  Use line height for text that spans multiple lines. Text on a single line
  (like a heading or CTA) should not set a line height.
</Alert>

- `--line-height-heading-1`
- `--line-height-heading-2`
- `--line-height-heading-3`
- `--line-height-heading-4`
- `--line-height-heading-5`
- `--line-height-body-16`
- `--line-height-body-14`
- `--line-height-body-small`
